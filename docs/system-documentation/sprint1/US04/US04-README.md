# US04 - As an Editor, I want to create a scenario for a selected map.

As an Editor, I want to create a scenario for a selected map.

## 3. Overview

This user story allows an Editor to create a scenario for a specific map. A scenario represents a historical period with technological restrictions and historical events that affect gameplay.

### Key Features:
- Select a map from existing maps
- Define a scenario name, start date, and end date
- Select available locomotive types for the scenario
- Select available industry types for the scenario
- Add historical events that will impact gameplay
- Save the scenario for future gameplay

### Dependencies:
- Requires existing maps to be available in the system
- Requires locomotive types and industry types to be defined
- Editor must be logged in to create scenarios
