[# Glossary

**Terms, Expressions and Acronyms (TEA) must be organized alphabetically.**

Column "TEA (PT)" is optional. If the team does not intend to use it, then remove it.

| **_TEA_** (EN)                | **_TEA_** (PT)               | **_Description_** (EN)                                                                                           |
|:------------------------------|:-----------------------------|:-----------------------------------------------------------------------------------------------------------------|
| **Acceptance Criteria**       | **Critérios de Aceitação**   | Specific conditions that must be met for a user story to be considered complete and acceptable.                  |
| **Building**                  | **Edifício**                 | An upgrade that enhances a station’s performance (e.g., telegraph, customs, hotel).                              |
| **BuildingType**              | **Tipo de Edifício**         | Classification of station upgrades (e.g., Telegraph, Hotel, Silo).                                               |
| **Cargo**                     | **Carga**                    | The goods or resources transported by trains between stations.                                                   |
| **CargoType**                 | **Tipo de Carga**            | The type of goods being transported (Passengers, Mail, Coal, etc.).                                              |
| **Carriage**                  | **Vagão**                    | A rail vehicle used to transport goods or passengers.                                                            |
| **City**                      | **Cidade**                   | A populated area on the map that generates and consumes passengers, mail, and final products.                    |
| **Demand**                    | **Procura**                  | The level of need for a specific cargo type at a station, ranging from 0-9, affecting payment values.            |
| **Depot**                     | **Depósito**                 | Smallest station type (3x3 radius, lowest cost).                                                                 |
| **Economic Radius**           | **Raio Económico**           | Area around a station where industries/cities contribute cargo demand/supply.                                    |
| **Editor**                    | **Editor**                   | Person responsible for creating and managing maps and scenarios in the system.                                   |
| **Electrified Line**          | **Linha Eletrificada**       | Railway line that supports electric locomotives.                                                                 |
| **Graphviz/GraphStream**      | **Graphviz/GraphStream**     | Software packages used for network visualization of stations and railway connections.                            |
| **House Block**               | **Bloco Habitacional**       | A unit representing residential housing within a city.                                                           |
| **Industry**                  | **Indústria**                | A general term for any industry (Primary, Transforming, or Mixed).                                               |
| **IndustryType**              | **Tipo de Indústria**        | Classification of industries (Mine, Farm, Steel Mill, etc.).                                                     |
| **JaCoCo**                    | **JaCoCo**                   | A Java code coverage tool used to measure the effectiveness of unit tests.                                       |
| **JUnit 5**                   | **JUnit 5**                  | A Java testing framework used for unit testing.                                                                  |
| **Jupyter Notebook**          | **Jupyter Notebook**         | Development environment used for statistical analysis and data visualization tasks.                              |
| **Location**                  | **Localização**              | Represents the X, Y coordinates of an element on the map.                                                        |
| **Locomotive**                | **Locomotiva**               | A powered rail vehicle that provides the force to move a train.                                                  |
| **LocomotiveType**            | **Tipo de Locomotiva**       | Classification of locomotives (Steam, Diesel, Electric).                                                         |
| **Map**                       | **Mapa**                     | A predefined area where cities, industries, and railway lines are placed.                                        |
| **Mixed industries**          | **Industria mista**          | Import or export any cargo.                                                                                      |
| **MVP**                       | **MVP**                      | Minimum Viable Product - the basic version of the software with essential features for initial release.          |
| **Path**                      | **Caminho**                  | A collection of connected segments that define a train's route.                                                  |
| **Player**                    | **Jogador**                  | Person who wants to play the game.                                                                               |
| **Point-of-Route**            | **Ponto deRota**             | A specific station in a train's route where cargo loading/unloading operations are defined.                      |
| **Port**                      | **Porto**                    | A type of mixed industry that can import, export, and transform various types of cargo.                          |
| **Primary sector industries** | **Industria setor primário** | Generate resources, such as iron, coal, bauxite or vegetables, cereals,wool                                      |
| **Product Owner**             | **Proprietário do Produto**  | Person representing the organization developing the game, responsible for defining requirements and priorities.  |
| **Railway Line**              | **Linha Ferroviária**        | A connection between two stations that allows train movement.                                                    |
| **Railway Network**           | **Rede Ferroviária**         | The system that manages trains, stations, and railway operations.                                                |
| **RDD**                       | **RDD**                      | Requirement-Driven Development, a software development methodology that focuses on the product's attributes.     |
| **Route**                     | **Rota**                     | A predefined path a train follows between stations, determining cargo pickup and delivery points.                |
| **Scenario**                  | **Cenário**                  | A set of conditions, restrictions, and historical contexts applied to a selected map.                            |
| **SCRUM**                     | **SCRUM**                    | Agile methodology used for project management and iterative development.                                         |
| **Scrum Master**              | **Scrum Master**             | Team member responsible for facilitating the SCRUM process during each Sprint (rotative role).                   |
| **Simulator**                 | **Simulador**                | The software component that models railway operations based on user-defined scenarios.                           |
| **Sprint**                    | **Sprint**                   | A four-week development cycle in the SCRUM framework.                                                            |
| **Station**                   | **Estação**                  | A location on the map where passengers and goods can be loaded or unloaded.                                      |
| **StationType**               | **Tipo de Estação**          | Classification of stations (Depot, Station, Terminal).                                                           |
| **Supply**                    | **Oferta**                   | The amount of cargo generated by industries within a station's economic radius.                                  |
| **TDD**                       | **TDD**                      | Test-Driven Development, a software development methodology that emphasizes writing tests before implementation. |
| **Terminal**                  | **Terminal**                 | Largest station type (5x5 radius, highest cost).                                                                 |
| **Train**                     | **Comboio**                  | A composition of a locomotive and carriages used for transportation of passengers or cargo.                      |
| **Transforming industries**   | **Industria transformadora** | Transform the resources provided by the Primary Sector Industries into new products.                             |
